２．機能・構成
アプリの概要：
１．Webアプリの概要
「北海道 4大都市の人口比較分析」
札幌、旭川、函館、帯広の4つの都市の2020年の人口を比較した。また、グラフだけでなく、気づいたことを書く「考察」欄を設けた。

データ確認：
2020年の4都市の人口データ

データ可視化：
棒グラフ：都市ごとの人口数を表示。
円グラフ：各都市の人口の占有率を可視化した。
メトリック表示：各都市の人口を大きく表示させた
